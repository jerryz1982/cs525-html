Interactive control of PiGuard

Beginning this week, PiGuard agents (Raspberry Pi) can be controlled by RestFul APIs, enabling the management of Pi agents with a click of a button on web dashboard on desktop PC or a mobile APP. The actions which user can perform are:

  • Show all the agents
  • Delete an agent
  • Arm/Disarm an agent
  • Mute a sound alert
  • Reboot an/all agent(s)
Users with coding skills can create their own logic referencing the API reference documentation.

Social media integration

Twitter has been supported as a notification channel for security events. The tweet, which PiGuard will pose includes timestamp of the event, event type and a photo taken by the web cam if motion is detected by the PIR motion sensor.

The social media integration is a plugin-based architecture, so more and more social media sites such as Instagram, Facebook, Wechat can be supported. Users can also develop their own plugin as they may find useful.
